Guide pour le client

Bienvenue dans l'application ManiocAGRI ! Ce guide vous aidera à utiliser la plateforme facilement.

🎯 Objectif

Permettre au client (producteur, acheteur, ou autre utilisateur) de se familiariser rapidement avec les fonctionnalités principales de ManiocAGRI.

🔐 Connexion et authentification

  1. Ouvrez votre navigateur et accédez à l'URL fournie par votre administrateur.
  2. Sur la page de connexion (authentification.html), entrez vos identifiants (email et mot de passe).
  3. Cliquez sur Se connecter.
  4. En cas d’oubli, utilisez l’option de réinitialisation de mot de passe si elle est disponible, ou contactez le support.

🛒 Consultation du catalogue

  • Rendez-vous sur catalogue.html pour voir la liste des produits disponibles.
  • Vous pouvez filtrer ou rechercher des produits selon vos besoins (si ces options sont implémentées).
  • Cliquez sur un produit pour afficher ses détails sur la page produit-details.html.

📝 Passer une commande

  1. Sur la page de détails d’un produit, sélectionnez la quantité souhaitée.
  2. Cliquez sur Ajouter au panier (ou équivalent).
  3. Accédez à votre panier et vérifiez les articles.
  4. Choisissez un mode de paiement dans la liste : Mobile Yass, Visa ou Moov Money. Lorsque vous sélectionnez un mode de paiement, le montant total s'affiche automatiquement.
  5. Cliquez sur Confirmer la commande pour la valider. Le système enregistre le paiement automatiquement si un mode est choisi.

💳 Modes de paiement disponibles

  • Mobile Yass – Transfert via Yass ToGo.
  • Visa – Carte bancaire.
  • Moov Money – Paiement mobile Moov.

Lorsque vous choisissez un mode de paiement, la commande est automatiquement marquée comme payée. L'administrateur pourra voir le statut de paiement et le moyen choisi.

📋 Suivi de commande

Vous pouvez consulter le statut de vos commandes dans l’interface utilisateur (par exemple via tableau de bord.html ou une section tracking.html).

Pour toute question sur un suivi, assurez-vous d’avoir le numéro de commande sous la main.

👥 Gestion de votre profil

Accédez à votre espace utilisateur pour mettre à jour vos informations (nom, adresse, mot de passe, etc.).

Si vous rencontrez des problèmes de connexion ou de paramètres, contactez le service client.

📞 Support et assistance

Pour toute aide supplémentaire, envoyez un e-mail à l’adresse de support ou appelez le numéro indiqué sur la page d’accueil ou dans la documentation fournie par votre fournisseur.

Conservez vos identifiants et vos informations de commande en cas de besoin de support.

Ce guide couvre les étapes de base pour utiliser la plateforme. Si de nouvelles fonctionnalités sont ajoutées, référez-vous à la documentation mise à jour ou contactez votre gestionnaire.

Bonne utilisation et merci d’avoir choisi ManiocAGRI ! 🌱
